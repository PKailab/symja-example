## SymbolQ

```
SymbolQ(x)
```

> is `True` if `x` is a symbol, or `False` otherwise.

### Examples

```
>> SymbolQ(a)
True

>> SymbolQ(1)
False

>> SymbolQ(a + b)
False
```
