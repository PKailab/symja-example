## Minors

```
Minors(matrix) 
```

> returns the minors of the matrix.
	
See:
* [Wikipedia - Minor (linear algebra)](https://en.wikipedia.org/wiki/Minor_(linear_algebra))
	
### Examples

```
>> Minors({{1,4,7},{3,0,5},{-1,9,11}})
{{-12,-16,20},{13,18,-19},{27,38,-45}}
```
