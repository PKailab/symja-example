## NumberQ

```
NumberQ(expr)
```

> returns `True` if `expr` is an explicit number, and `False` otherwise.

### Examples

```
>> NumberQ(3+I)
True
 
>> NumberQ(5!)
True
 
>> NumberQ(Pi)
False
```
