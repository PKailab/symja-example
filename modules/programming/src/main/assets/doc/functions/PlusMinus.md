## PlusMinus

```
PlusMinus(a, b, ...)

a ± b ± ...
```

or

```
PlusMinus(a)

± a
```

> has no built-in evaluating function, but represents the structure of the `±` operator.

See 
* [Wikipedia - Plus-minus sign](https://en.wikipedia.org/wiki/Plus%E2%80%93minus_sign)

### Examples

```
>> \[PlusMinus] x
±x

>> x \[PlusMinus] y
x±y
```
