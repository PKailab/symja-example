## NumericQ

```
NumericQ(expr)
```

> returns `True` if `expr` is an explicit numeric expression, and `False` otherwise.

### Examples

```
>> NumericQ(E+Pi)
True
 
>> NumericQ(Sqrt(3))
True
```
