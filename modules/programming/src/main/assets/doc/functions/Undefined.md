## Undefined

```
Undefined 
```

> represents an undefined result for example in the `ConditionalExpression` function.
 
### Examples

If the condition in `ConditionalExpression` is `False`, `Undefined` will be returned:

``` 
>> ConditionalExpression(Sin(x)^2, 42==1)
Undefined
```
