## List

```
List(e1, e2, ..., ei)
```

or

```
{e1, e2, ..., ei}
```

> represents a list containing the elements `e1...ei`.

### Examples

`List` is the head of lists:

```
>> Head({1, 2, 3})
List
```

Lists can be nested:

```
>> {{a, b, {c, d}}}
{{a, b, {c, d}}}
```
